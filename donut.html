🍩 Interactive 3D Donut Experience

A web-based 3D application that allows users to explore realistic donut models in real time using interactive rendering.

Overview

Problem

Most web experiences are static and lack interactive depth, limiting user engagement with visual content.

Goal

Create an interactive 3D web experience that is visually engaging while maintaining strong performance.

Role

3D Modeling (Blender)
Frontend Development (Three.js)
UI & Interaction Design

Process

3D Modeling

Built donut models using a torus base and refined shapes with sculpting tools to achieve realistic imperfections.

Materials & Texturing

Applied realistic materials and lighting responses using Blender’s shader editor to simulate different flavors.

Optimization

Reduced polygon counts and exported models as .glb files to ensure smooth performance in the browser.

Web Integration

Implemented real-time rendering, lighting variations, and wireframe modes using Three.js.

Key Features

Real-time 3D rendering
Interactive lighting modes
Wireframe visualization
Smooth animation
Optimized performance

Challenges

A major challenge was balancing visual quality with performance. Optimizing models while preserving realism required careful adjustments to geometry and lighting.

Final Outcome

Reflection

This project strengthened my understanding of real-time rendering and performance optimization. In future iterations, I would add user-controlled camera interactions and explore product customization features.
